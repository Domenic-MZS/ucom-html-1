##  Prerequisites
 - Docker 24.x

##  How to Get Start? - Development Server
To serve the project on port 3000 run:
```bash
  make build run
```

Clean Steps:
```bash
  make clean 
```

Start a terminal:
```bash 
  make terminal
```

Clean start with:
```bash
  make clean build run
```
